# 小红书文案生成 Bot

通过 QQbot 命令操作，帮你自动生成 AI 漫剧领域的小红书爆款文案。

---

## 📱 手机 QQ 可用命令

| 命令 | 功能 | 示例 |
|------|------|------|
| `写文案` | 基于最近搜集内容写文案 | `写文案` |
| `写文案 [主题]` | 指定主题写文案 | `写文案 AI 漫剧教程` |
| `写标题` | 生成 10 个爆款标题 | `写标题 AI 漫剧` |
| `优化文案` | 优化已有文案 | `优化文案 [粘贴文案]` |
| `标签推荐` | 推荐热门标签 | `标签推荐 AI 漫剧` |
| `文案模板` | 查看文案模板 | `文案模板` |

---

## 🎯 爆款文案公式

### 标题公式

| 类型 | 公式 | 示例 |
|------|------|------|
| **悬念式** | "XX 天，我从 0 到 XX" | "7 天，我从 0 做出第一部 AI 漫剧" |
| **对比式** | "以为 XX，结果 XX" | "以为 AI 漫剧很难，结果 3 小时搞定" |
| **数字式** | "X 个 XX，让你 XX" | "5 个 AI 工具，让你轻松做漫剧" |
| **情绪式** | "太 XX 了！XX 必看" | "太香了！AI 漫剧小白必看" |
| **教程式** | "手把手教你 XX" | "手把手教你用 AI 做漫剧" |
| **揭秘式** | "XX 背后的秘密" | "AI 漫剧爆款背后的秘密" |

### 正文结构

```
【开头】情绪/痛点/悬念 (20-30 字)
【中间】价值输出/教程/故事 (100-200 字)
【结尾】互动引导/关注引导 (20-30 字)
【标签】5-10 个相关标签
```

## 🛠️ 脚本说明

### writer.mjs

主文案生成脚本。

```bash
node scripts/writer.mjs [主题] [风格]
```

风格可选：教程、分享、测评、种草、干货

---

## ⚠️ 注意事项

1. **文案长度**：小红书正文建议 100-300 字
2. **标签数量**：5-10 个为宜，不要超过 15 个
3. **互动引导**：结尾一定要有互动引导
4. **表情符号**：适当使用 emoji 增加可读性
