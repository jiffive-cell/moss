# AI 漫剧运营总控 Bot

🎬 **一站式 AI 漫剧小红书运营助手**

通过 QQbot 命令操作，整合搜集、文案、配图、发布 4 个 Bot，让你在手机上就能完成整个小红书运营流程。

---

## 📱 手机 QQ 可用命令

### 快速命令

| 命令 | 功能 | 示例 |
|------|------|------|
| `今天发什么` | 推荐今日发布内容 | `今天发什么` |
| `一键生成` | 自动生成完整内容 | `一键生成 AI 漫剧教程` |
| `发布` | 发布最近生成的内容 | `发布` |
| `运营日报` | 查看今日运营数据 | `运营日报` |
| `帮我运营` | 启动自动运营流程 | `帮我运营` |

### 分步命令

| 命令 | 功能 | 对应 Bot |
|------|------|---------|
| `搜漫剧` | 搜索行业资讯 | 🔍 搜集 Bot |
| `爆款数据` | 抓取爆款数据 | 🔍 搜集 Bot |
| `写文案` | 生成小红书文案 | ✍️ 文案 Bot |
| `写标题` | 生成爆款标题 | ✍️ 文案 Bot |
| `生成配图` | 生成封面配图 | 🎨 配图 Bot |
| `封面` | 生成小红书封面 | 🎨 配图 Bot |
| `发布` | 发布到小红书 | 📤 发布 Bot |
| `定时发布` | 设置定时发布 | 📤 发布 Bot |

### 管理命令

| 命令 | 功能 |
|------|------|
| `运营状态` | 查看各 Bot 状态 |
| `定时搜集` | 设置自动搜集 |
| `发布计划` | 查看发布计划 |
| `数据报表` | 查看详细数据 |
| `重置` | 重置运营状态 |

## ⚠️ 注意事项

1. **主动消息权限**：定时推送需要 QQbot 主动消息权限
2. **API 限额**：生图 API 有每日免费限额
3. **发布频率**：建议每天 1-3 篇，避免频繁发布
4. **内容质量**：确保原创，避免抄袭
