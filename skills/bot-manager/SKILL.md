# Bot 团队管理中心

🎛️ **一站式管理你的 AI 漫剧运营 Bot 团队**

通过 QQbot 命令操作，查看和管理所有 Bot 的工作状态、任务历史、配置信息。

---

## 📱 手机 QQ 可用命令

### 总览命令

| 命令 | 功能 | 示例 |
|------|------|------|
| `bot 列表` | 查看所有 Bot | `bot 列表` |
| `bot 状态` | 查看整体状态 | `bot 状态` |
| `运营状态` | 查看运营概览 | `运营状态` |

### 单独查看 Bot

| 命令 | 查看对象 | 示例 |
|------|---------|------|
| `查看 搜集 Bot` | 搜集 Bot 状态 | `查看 搜集 Bot` |
| `查看 文案 Bot` | 文案 Bot 状态 | `查看 文案 Bot` |
| `查看 配图 Bot` | 配图 Bot 状态 | `查看 配图 Bot` |
| `查看 发布 Bot` | 发布 Bot 状态 | `查看 发布 Bot` |
| `查看 [名字]` | 查看指定 Bot | `查看 ai-manhua-search` |

### 管理命令

| 命令 | 功能 | 示例 |
|------|------|------|
| `启动 [Bot]` | 启动 Bot | `启动 搜集 Bot` |
| `停止 [Bot]` | 停止 Bot | `停止 搜集 Bot` |
| `重启 [Bot]` | 重启 Bot | `重启 文案 Bot` |
| `配置 [Bot]` | 查看/修改配置 | `配置 配图 Bot` |
| `任务历史 [Bot]` | 查看任务记录 | `任务历史 搜集 Bot` |
| `清空缓存 [Bot]` | 清空 Bot 缓存 | `清空缓存 文案 Bot` |

### 数据命令

| 命令 | 功能 | 示例 |
|------|------|------|
| `今日任务` | 查看今日任务 | `今日任务` |
| `任务统计` | 查看任务统计 | `任务统计` |
| `数据报表` | 查看详细数据 | `数据报表` |
| `导出日志` | 导出运行日志 | `导出日志` |

## 📊 数据看板

### 健康度评分

| 评分 | 状态 | 说明 |
|------|------|------|
| 90-100 | 🟢 优秀 | 所有 Bot 正常运行 |
| 70-89 | 🟡 良好 | 大部分 Bot 正常 |
| 50-69 | 🟠 一般 | 部分 Bot 异常 |
| <50 | 🔴 警告 | 多数 Bot 异常 |

### 统计维度

| 维度 | 指标 |
|------|------|
| **运行状态** | 在线/离线/异常 |
| **任务统计** | 次数、成功率、平均耗时 |
| **资源占用** | 内存、CPU、存储 |
| **内容产出** | 文案数、图片数、发布数 |
| **错误追踪** | 错误类型、频率、原因 |

## 🛠️ 脚本说明

### manager.mjs

主管理脚本。

```bash
node scripts/manager.mjs [命令] [Bot 名]
```

### monitor.mjs

监控脚本，定期检查 Bot 健康状态。

```bash
node scripts/monitor.mjs
```

---

## ⚠️ 注意事项

1. **权限控制**：管理命令需要管理员权限
2. **日志保留**：任务历史保留 30 天
3. **缓存清理**：建议每周清理一次缓存
4. **健康检查**：每天自动检查 Bot 状态
