# AI 漫剧配图生成 Bot

通过 QQbot 命令操作，帮你自动生成 AI 漫剧内容的小红书封面图、配图。

---

## 📱 手机 QQ 可用命令

| 命令 | 功能 | 示例 |
|------|------|------|
| `生成配图 [描述]` | 根据描述生成图片 | `生成配图 AI 漫剧封面 赛博朋克` |
| `封面 [主题]` | 生成小红书封面 | `封面 AI 漫剧教程` |
| `换风格 [风格]` | 切换生成风格 | `换风格 动漫风` |
| `图生图 [图片]` | 基于图片生成 | `图生图 [上传图片]` |
| `批量生成 [数量]` | 批量生成多张 | `批量生成 4 张` |

---

## 🎨 支持的风格

| 风格 | 说明 | 适用场景 |
|------|------|---------|
| **动漫风** | 日式动漫风格 | 漫剧封面、角色图 |
| **赛博朋克** | 霓虹灯、未来感 | 科幻题材 |
| **国潮风** | 中国风 + 现代 | 古风漫剧 |
| **简约风** | 简洁、高级感 | 教程类封面 |
| **插画风** | 手绘插画感 | 故事类内容 |
| **3D 渲染** | 立体、精致 | 高质量封面 |

## 🛠️ 脚本说明

### generate.mjs

主生图脚本（需配置 API）。

```bash
node scripts/generate.mjs "[描述]" [风格] [尺寸]
```

---

## ⚠️ 注意事项

1. **API 配置**：需要先配置通义万相 API
2. **生成时间**：每张图约 20-40 秒
3. **图片保存**：生成的图片保存在 `workspace/generated/` 目录
4. **使用限制**：免费 API 有每日限额
