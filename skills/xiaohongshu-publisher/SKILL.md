# 小红书自动发布 Bot

通过 QQbot 命令操作，帮你自动发布 AI 漫剧内容到小红书。

---

## 📱 手机 QQ 可用命令

| 命令 | 功能 | 示例 |
|------|------|------|
| `发布` | 发布最近生成的内容 | `发布` |
| `发布 [文案]` | 发布指定文案 | `发布 [文案内容]` |
| `定时发布 [时间]` | 设置定时发布 | `定时发布 今天晚上 8 点` |
| `发布状态` | 查看发布状态 | `发布状态` |
| `登录小红书` | 登录小红书账号 | `登录小红书` |

---

## 🤖 QQbot 交互流程

### 1. 快速发布

**用户**: `发布`

**Bot 回复**:
```
📤 准备发布到小红书...

【待发布内容】
• 文案：已准备 (最近生成的文案)
• 图片：已准备 (最近生成的配图)
• 标签：已准备

⚠️ 发布前确认：
1. 文案是否满意？回复"改文案"可修改
2. 配图是否满意？回复"换图"可更换
3. 确认发布？回复"确认发布"

💡 或直接回复"确认发布"一键发布~
```

**用户**: `确认发布`

**Bot 回复**:
```
📤 正在发布到小红书...

[1/4] 登录小红书... ✅
[2/4] 上传图片... ✅
[3/4] 填写文案... ✅
[4/4] 点击发布... ✅

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ 发布成功！

📊 预计数据：
• 曝光：500-2000
• 点赞：50-200
• 收藏：20-80

💡 我会持续追踪数据，明天给你汇报~

🔗 查看笔记：https://xiaohongshu.com/...
```

### 2. 定时发布

**用户**: `定时发布 今天晚上 8 点`

**Bot 回复**:
```
⏰ 好的，今天晚上 8 点自动发布到小红书~

【发布计划】
• 时间：今天 20:00
• 内容：AI 漫剧教程
• 配图：已准备

💡 最佳发布时间：
• 工作日：12:00-13:00, 18:00-21:00
• 周末：10:00-12:00, 19:00-22:00

说"取消发布"可取消定时~
```

## ⚠️ 注意事项

1. **登录状态**：首次需要手动扫码登录
2. **发布频率**：建议每天 1-3 篇，避免频繁发布
3. **内容质量**：确保文案和图片质量，避免低质内容
4. **平台规则**：遵守小红书社区规范
