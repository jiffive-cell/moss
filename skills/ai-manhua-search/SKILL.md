# AI 漫剧资讯搜集 Bot

通过 QQbot 命令操作，帮你自动搜集 AI 漫剧行业资讯、爆款数据、观众喜好分析。

---

## 📱 手机 QQ 可用命令

| 命令 | 功能 | 示例 |
|------|------|------|
| `搜漫剧` | 搜索最新 AI 漫剧资讯 | `搜漫剧` |
| `爆款数据` | 抓取小红书/抖音爆款数据 | `爆款数据` |
| `行业动态` | 搜集行业最新动态 | `行业动态` |
| `大模型动态` | AI 大模型最新进展 | `大模型动态` |
| `观众喜好` | 分析观众喜好趋势 | `观众喜好` |
| `日报` | 生成今日汇总日报 | `日报` |
| `定时搜集` | 设置每天自动搜集 | `定时搜集 每天早上 9 点` |

## 🛠️ 脚本说明

### search.mjs

主搜索脚本，调用 searxng 和 agent-browser 搜集数据。

```bash
node scripts/search.mjs [关键词] [平台]
```

### daily-report.mjs

生成日报脚本，整理当天所有数据。

```bash
node scripts/daily-report.mjs
```

---

## ⚠️ 注意事项

1. **搜索频率**：避免短时间内大量搜索，防止被限流
2. **数据缓存**：搜集的数据会缓存 24 小时，避免重复搜索
3. **主动消息**：定时推送需要 QQbot 主动消息权限
